# Programozási Tételek

## Elõkészítés
 - 10 elemû, egész számot tartalmazó tömb létrehozása
 - A tömb elemei: 
      - 12, 7, 2, 4, 45, 99, 1, 8, 95, 50
 - A tömb az összes metódusban látható legyen
 - A tombkiiras() metódus létrehozása
     - nem ad vissza értéket
     - nincs paramétere
     - a tömb elemeit vesszõvel elválasztva egy sorban a console-ra írja

## Összegzés
 - Az osszegzes() metódus létrehozása
     - összegzés tétele a tömbön
     - eredmény kiírása a console-ra

## Megszámolás
 - A megszamolas() metódus létrehozása
     - írassuk ki hány 3-mal osztható szám van a tömbben
     - eredmény kiírása a console-ra

## Eldöntés
 - Az eldontes() metódus létrehozésa
     - van-e a tömbben 69?
     - kiírni, hogy van/nincs

## Kiválasztás
 - A kivalasztas() metódus létrehozása
     - hányadik az 50-es érték?
     - kiírni a helyét a tömbben

## Keresés (lineáris)
 - A kereses() metódus létrehozása
     - van-e benne 50, és ha van hányadik?
